Study Dashboard 📚

A modern and interactive Study Dashboard designed to help students organize their academic life, track progress, manage study sessions, and improve productivity.

🚀 Overview

The Study Dashboard is a student-focused productivity platform built to simplify learning management. It allows users to monitor GPA progress, manage daily tasks, organize courses, and stay consistent with study goals.

This project was created to combine technology with education and demonstrate practical programming and web development skills.


---

✨ Features

📅 Daily Study Planner

✅ Task & Assignment Tracker

📊 GPA Calculator

⏰ Study Session Timer

📈 Progress Analytics Dashboard

🌙 Dark/Light Mode UI

🔐 User Login & Authentication

📚 Course Management System

🎯 Goal Tracking System

📱 Responsive Design for Mobile and Desktop
